This is the code file for paper 
"PREDICTING BATTERY LIFETIME UNDER VARYING USAGE CONDITIONS FROM EARLY AGING DATA"

The raw feature data files are in the "train_test_split" folder.
Run hbm_final_CRPS.ipynb first, then you gonna have all other results recordings for plotting
Run hbm_plots_CRPS.ipynb for the figures in the paper.


The key python packages used:
numpy
pandas
matplotlib
scikit-learn
pymc3
